‚ö°js/web-components.min.js

./js/web-components.min.js
Purpose: Stoplight Elements API component library for interactive documentation

What is Stoplight Elements?

Stoplight Elements is a collection of rich, embeddable web components powered by OpenAPI specifications. It's developed by Stoplight, a leading API design and documentation platform.

Key Features:

  • <elements-api>: Main documentation component that renders complete API docs
  • Interactive Examples: Try-it-out functionality with real API calls
  • Schema Browser: Visual representation of data models and validation
  • Code Generation: Automatic code samples in multiple languages
  • Responsive Design: Mobile-friendly and customizable themes
  • Search & Navigation: Built-in search and organized endpoint navigation
// Configuration options for Elements API <elements-api id="docs" hideSchemas="true" // Simplify interface router="hash" // Enable deep linking layout="sidebar" // Navigation layout logo="[logo-url]" // Custom branding > </elements-api>

Why Stoplight Elements?

  • Industry Standard: Used by thousands of companies worldwide
  • OpenAPI Native: Built specifically for OpenAPI 3.0 specifications
  • Zero Configuration: Works out-of-the-box with any valid OpenAPI spec
  • Extensible: Highly customizable with CSS and JavaScript
  • Performance: Optimized for large API specifications
