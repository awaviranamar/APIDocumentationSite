üåü What is Stoplight Elements?

Stoplight Elements is a collection of rich, embeddable web components powered by OpenAPI specifications. Created by Stoplight, a leading API design and documentation platform, Elements provides everything you need to create world-class API documentation.

Key Philosophy

Elements follows the principle of "OpenAPI-first documentation" - meaning your API specification is the single source of truth, and beautiful documentation is automatically generated from it.

üé® Beautiful by Default

Stunning, professional-looking documentation with zero configuration required. Just point it at your OpenAPI spec and you're done.

üîß Highly Customizable

Extensive theming options, custom CSS support, and configuration properties to match your brand perfectly.

üì± Mobile-First

Responsive design that works flawlessly on desktop, tablet, and mobile devices with touch-friendly interactions.

‚ö° Performance Optimized

Built for large API specifications with thousands of endpoints. Lazy loading and efficient rendering ensure fast performance.

üîÑ 2. Processing Layer
